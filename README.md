# Sistema de Chamados Internos

Sistema interno desenvolvido em Flask para:

- abertura de chamados
- dashboard TI
- controle de ativos
- mapas operacionais
- chat interno
- exportação PDF/XLSX

## Tecnologias

- Python
- Flask
- PostgreSQL
- Docker
- HTML/CSS
- Chart.js

---

# Executar

## Docker

```bash
docker compose up --build
```

Sistema:
http://localhost:5000

---

## Banco

PostgreSQL via Docker.

Variáveis configuradas no `.env`.
